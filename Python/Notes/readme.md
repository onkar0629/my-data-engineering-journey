## 🐍 What is Python?

Python is a **general-purpose, high-level, interpreted programming language** developed by **Guido van Rossum** and first released in **1991**.

It supports multiple programming paradigms, making it one of the most versatile programming languages used in software development, data engineering, artificial intelligence, web development, automation, and many other domains.

---

## ✨ Characteristics of Python

Python supports multiple programming paradigms:

- 🔹 Functional Programming
- 🔹 Object-Oriented Programming (OOP)
- 🔹 High-Level Programming
- 🔹 Interpreted Language
- 🔹 Scripting Language

Because it supports all of these, Python is known as a **General-Purpose Programming Language**.

## ⚙️ Installing Python

### Step 1

Visit the official Python website:

> https://www.python.org/

Download the latest stable version suitable for your operating system.

### Step 2

Run the installer.

Before clicking **Install Now**, make sure to check:

✅ **Add Python to PATH**

Then click **Install Now**.

### Step 3

Wait until the installation completes successfully.

---

## ✅ Verify Python Installation

Open Command Prompt or Terminal and run:

```bash
python --version
```

or

```bash
python3 --version
```

If Python is installed correctly, it will display the installed version.

Example:

```bash
Python 3.13.5
```

## 💻 Ways to Write Python Programs

Python programs can be written in three different ways.

### 1️⃣ Interactive Mode

- Executes one statement at a time.
- Immediate output.
- Best for learning and testing small snippets.

Example:

```python
>>> 10 + 20
30
```

---

### 2️⃣ Script Mode

- Write code inside a `.py` file.
- Save the file.
- Execute the complete program.

Example:

```python
print("Hello, Python!")
```

---
