## Overview

Python is a **high-level, interpreted, general-purpose programming language** developed by **Guido van Rossum**. It was first released in **1991** and has become one of the world's most popular programming languages because of its simple syntax, readability, and extensive ecosystem.

Python supports multiple programming paradigms such as Functional Programming, Object-Oriented Programming, Procedural Programming, and Scripting, making it suitable for everything from automation scripts to enterprise applications and Data Engineering.

# 1. What is Python?

Python is a **General-Purpose Programming Language** developed by **Guido van Rossum**.

It supports multiple programming paradigms:

- Functional Programming
- Object-Oriented Programming (OOP)
- Procedural Programming
- Scripting

Because Python supports all these paradigms, it is called a **General-Purpose Programming Language**.

> [!NOTE]
> Python's simple syntax makes it one of the easiest programming languages for beginners while still being powerful enough for large-scale applications.

---

# 2. Features of Python

### - Open Source

Python is free to download, use, and modify.

### - Easy to Learn

Python has clean and readable syntax.

```python
print("Hello, World!")
```

### - Extensive Standard Library

Python includes hundreds of built-in modules that simplify development.

### - Interpreted Language

Python executes code **line by line**, making debugging easier.

### - Dynamically Typed

Variable types are determined automatically.

```python
name = "Onkar"
age = 22
salary = 50000.50
```

### - Platform Independent

Python programs run on Windows, macOS, and Linux with little or no modification.

### - Large Community Support

Python has one of the largest developer communities in the world.

# 3. Installing Python

Follow these steps:

1. Visit the official Python website.
2. Download the latest stable version.
3. Run the installer.
4. Select **Add Python to PATH**.
5. Click **Install Now**.

> [!IMPORTANT]
> Always enable **Add Python to PATH** during installation.

---

# 4. Verify Python Installation

Open Terminal (macOS/Linux) or Command Prompt (Windows).

Run:

```bash
python --version
```

or

```bash
python3 --version
```

If Python is installed correctly, the installed version will be displayed.

# 5. Working with Python

Python programs can be executed in three ways.

### Interactive Mode

- Executes one statement at a time.
- Produces immediate output.
- Useful for testing small snippets.

### Scripting Mode

- Write code inside a `.py` file.
- Save the file.
- Execute whenever required.

### IDE (Integrated Development Environment)

Popular IDEs include:

- PyCharm
- Visual Studio Code
- Jupyter Notebook
- Google Colab

For this repository, we'll use **PyCharm**.

# 10. Common Mistakes

- Forgetting to add Python to PATH.
- Saving files without the `.py` extension.
- Naming files with spaces.
- Confusing Interactive Mode with Scripting Mode.

> [!WARNING]
> Avoid naming your Python files `python.py`, `random.py`, or `math.py`, as they can conflict with Python's built-in modules.
